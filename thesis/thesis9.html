The Poisson Process: Meaning, Properties, Simulations

In statistics a Poisson Point Process is a kind of random mathematical object. It consists of points randomly located on a mathematical space, where each point is indipendent from each other. This is often called Poisson random measure or, simply, Poisson process. This process is characterized via the Poisson distribution, where the probability is dipended on N (called a Poisson random variable). Therefore, the probability of non-overlapping regions is: $$ Pr\left\{ N = n \right\}=\frac{\lambda^{n}}{n!}e^{-\lambda} $$ This is the probability that the Nth number is equal to n. The other variables are \( \lambda \) , the parameter of the Poisson distribution, \( e \) the Euler's constant and \( n \), which is the number of events that we are calculating. To recap, the properties that a Poisson Process follows are:
  • Indipendece.
  • Homogeneity , since the points are distributed uniformally throughout the space or time interval.
  • Poisson property
In particular, a Poisson Process is formally defined as follow:
Let \( \lambda > 0 \) be fixed. The couting process \( \{ N(t), t \in [0, \infty) \} \) is called a Poisson process with rates \( \lambda \) if the following conditions hold:
  1. \( N(0) = 0 \),
  2. \( N(t) \) has indipendent increments,
  3. The number of arrivals in any interval of length \( \tau > 0 \) has \( Poisson( \lambda \tau ) \) distribution.
A Poisson distribution is defined by the following formula: $$ P(X = k) = \frac{e^{- \lambda} \lambda^k}{k!}$$ It's possible to look at the Poisson distribution via the 5th assigned homework. An interesting property of a Poisson process involves figuring out how long we have to wait until the next event. We can consider the following scenario: A bus arrives at the bus-stop every 12 minutes on average. How long can we expect to wait to see the next bus if we arrive at a random time? The probability of waiting a a given amount of time decreases exponentially as time increases. This behaviour is described by the following equation: $$ \large{ P(T > t) } = \Large { e^{- \frac{\text{events}}{\text{time}} * t} } $$ We can see that the waiting time between events are memoryless, so the time between two events has no effect on the time between any other events. This property is the Markov property!
