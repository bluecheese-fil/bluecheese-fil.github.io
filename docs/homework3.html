The shape of the distributions will depend on the values chosen for N, M and P. When both M and N are large, but P is small, the distribution will tend to resemble a normal distribution. This is true since, when we posses a larger sample, the sampling distribution of the mean is normally distributed, even if your original variable is not normally distributed. This aproximation is based on the Central Limit Theorem. The histograms will look like a normal distribution when M is large. If N is increased, the histograms will contain more details for a given point, showing different variations on the graph. Second question

Recall briefly the definition and math notions relevant to "probability space" and make some simple examples, indicating among the triple of the space the meaning of each element in your particular example. If you wanted to model probabilistically the homework Exercise 1, explain what are the 3 sets of your probability space and their elements, in this case.

In porbability theory, a probability space is a mathematical construct that formalizes an experiment or a random process with a model. This is written as a triple (Ω, F, P). A probability space consists of three elements:
  1. Ω is the sample space. This rapresents the set of all possible outcomes of an experiment. An event is an element that must be contained in Ω.
  2. F is the event space. This corresponds to a subset of the sample space and classifies some events as "measurable" or "events of interest".
  3. P is a probability function. In particular it associates a number between 0 and 1 to each event, where 0 means that the chosen event will never realize, where 1 means that the event will always occur.
We could construct an example based on two coin tosses. All the possible outcomes are:
Ω = { HH, HT, TH, TT }.
A possible event of interest could be all the events where tail happend at least once:
F = { HT, TH, TT }.
If the coins are fair, than all the outcomes must have the same probability:
P(HH) = P(HT) = P(TH) = P(TT) = 1/4.
The chance that an event will fall into our events of interest is 3/4.
To create a model it would be possible to chose:
  • Ω is a set that contains all the possible result graphs given N and M.
  • F would be a subset of Ω that we have interest to study.
  • P is a probability function that each graph will occur. Note that this is strictly related with P.
